The criteria for how we evaluate the output code

## Prompt Accuracy / Basic Requirements
- Does the code run?
- Does it use only plain HTML, CSS, and JavaScript?
- Does it create a slot machine app?
- Does the app make fun of AI?

## App/Code Quality
- Do the slot machine mechanics work correctly?
- Is the app visually appealing?
- How cleverly did the app make fun of AI?
- Is the code readable?
- Are there clear comments that explain how things work, but also don't overexplain?
- Are there any noticeable bugs?
